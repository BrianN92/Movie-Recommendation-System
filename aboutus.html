Decision Support System

Decision support system (DSS) is a computerized information system used to support decision-making in an organization or a business. It lets users sift through and analyze data and information that can be used to solve problems and make better decisions.

About Us

Our Project

A recommender System is one kind of DSS based on the users’ “rating” or “preference”, which has become increasingly popular in recent years, and is utilized in a variety of areas including movies, music, news, books, research and so on.

So, in this project, we decide to establish a entertainment recommender system based on cloud computing to help users choose what they like (movies, musics, ...).

    Project Component

    Dataset MovieLens 100K Datasetr

    • 100,000 ratings (1-5) from 1000 users on 1700 movies
    • Each user has rated at least 20 movies
    • Simple demographic info for the users (age, gender, occupation, zip)

    DigitalOcean - An IaaS Provider

    • They are a scalable compute platform with add-on storage, security, and monitoring capabilities to easily run production applications.
    • Deploy our VPS around the world in any of 8 datacenter regions.
    • Predictable, pay-as-you-go pricing.

    Flask - A micro web development framework for Python

    Features:

    • Contains development server and debugger
    • Integrated support for unit testing
    • Uses Jinja2 templating
    • Support for secure cookies (client side sessions)
    • Google App Engine compatibility

    Collaborative Filtering based on User Similarity

    • Build the user-item Rating Matrix
    • Build the Distance Matrix based on Cosine similarity
    • Predict the rating based on Distance Matrix by Weighted Average
